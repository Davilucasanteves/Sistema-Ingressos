# Tecnologias usadas para implementação
- JavaFX (Zulu JDK 21): plataforma e distribuição usadas para desenvolver a interface e executar a aplicação.
- SceneBuilder: ferramenta visual para construção das telas em FXML.
- VS Code: IDE utilizada para edição do código, organização do projeto e integração com extensões Java.
- Git: usado para controle de versão do código-fonte, criando um histórico rastreável de alterações e permitindo trabalhar com commits organizados.
- GitHub: utilizado para hospedagem do repositório, gerenciamento remoto, documentação no README e acompanhamento da evolução do projeto.
- StarUML: modelagem dos diagramas UML (classes, relações).
- Canva: criação dos elementos visuais e materiais de apresentação.
